벨만 포드
=
- 하나의 출발노드로부터 다른 노드까지의 최단 거리
- 음수 가중치 허용 / 음수 사이클 비허용(음수 사이클 확인 절차 거침)
- O(nm) 의 시간 복잡도를 가짐
- 동적 계획법(dp) + edge relaxation 기법

### 최단경로의 부분 경로는 역시 최단경로이다. 
- 최적해가 그 부분문제들의 최적해로 구성
- DP나 그리디 사용가능

### edge relaxation (변 경감)
- 기존의 거리보다 특정 노드를 경유해서 가는 비용이 작을 경우 업데이트

### 음수 사이클
- n - 1만큼 반복한 후 1회 더 반복
- 값이 변하면 음수사이클이 존재
